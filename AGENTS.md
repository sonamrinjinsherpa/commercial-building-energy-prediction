# Agent Instructions

This project is a Python-based data science and machine learning project using
the ASHRAE Great Energy Predictor III dataset.

## Guidelines

- Use Python 3.12 or newer.
- Follow PEP 8.
- Prefer type hints.
- Keep notebooks focused on one task.
- Use `pathlib` for filesystem paths.
- Do not commit datasets or generated model artifacts.
- Store raw data in `data/raw/`.
